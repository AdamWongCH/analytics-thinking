# Lesson: Real-World Analytics Systems

---

## Objective

Understand how analytics is applied in real-world contexts.

---

## The Common Mistake

Many learners think analytics is:

* running models
* producing outputs

But in reality:

* outputs are only useful if they drive decisions

---

## What is an Analytics System?

An analytics system includes:

1. Data
2. Processing
3. Analysis
4. Output
5. Decision

---

## Example

Student engagement system:

* data: logins, activity
* processing: aggregation
* analysis: identify risk
* output: dashboard
* decision: intervene

---

## Connecting Techniques

Different methods serve different roles:

* regression → understand relationships
* classification → predict risk
* clustering → identify segments
* association → detect patterns
* time series → monitor change
* NLP → understand feedback

---

## The Real Workflow

In practice:

1. Define the problem
2. Gather data
3. Analyse
4. Present insights
5. Enable decisions
6. Monitor outcomes

---

## Key Consideration

Analytics must:

* be usable
* be timely
* be trusted

---

## Key Takeaway

Analytics only matters if it changes decisions.
