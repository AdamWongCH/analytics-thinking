# 8. Time Series

---

## Learning Objectives

By the end of this module, you should be able to:

* understand what time series data is
* identify trends and patterns over time
* visualise time-based data
* interpret changes in behaviour

---

## Why This Module Matters

Many real-world problems involve time.

For example:

* student engagement over weeks
* sales over months
* system usage over time

Understanding patterns over time helps us:

* detect changes
* identify trends
* anticipate future behaviour

---

## The Core Idea

Time series analysis focuses on:

> how data changes over time

Time adds context that cannot be seen in static data.

---

## Scenario

You are given weekly student engagement data.

Your task is to:

* identify patterns over time
* interpret changes in behaviour
