# 1. Introduction to Data Thinking

## Learning Objectives

By the end of this module, you should be able to:

* Understand the difference between data and information
* Explain how data supports decision-making
* Describe the analytics value chain
* Recognise different types of analytics

---

## Core Idea

Data by itself has no meaning.

Only when data is processed and interpreted does it become useful for decision-making.

---

## The Analytics Flow

**Data → Information → Insight → Decision → Impact**

Most people stop at dashboards.

We go further — towards decisions and impact.
