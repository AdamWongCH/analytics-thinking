# Setup Guide

This course uses both **Python** and **R** for hands-on learning.

You do not need to be an expert in either tool to get started.

---

## Tools Used

* Python (e.g., pandas, scikit-learn)
* R (e.g., tidyverse, caret)

---

## Why both?

Each tool has its strengths:

* **Python** → widely used in machine learning and production systems
* **R** → strong in statistics and data exploration

Learning both gives you flexibility and perspective.
