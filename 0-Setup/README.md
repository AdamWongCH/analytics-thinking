# Setup Guide

This course uses Python and R for hands-on learning.

You do not need prior experience in both tools to get started.

---

## Tools Used

* Python for general data analysis and machine learning
* R for statistical analysis and data exploration

---

## Working Environment

Use the following environments:

### Python Labs

* Run using Google Colab or Jupyter Notebook

### R Labs

* Run using RStudio

---

## Getting Started

### For Python (Recommended: Google Colab)

* Open the notebook link provided in each lab
* No installation required

---

### For R (Recommended: RStudio)

1. Install:

   * R → https://cran.r-project.org
   * RStudio → https://posit.co/download/rstudio-desktop/

2. Open `.Rmd` files in RStudio

3. Click **Knit** to run the lab

---

## Key Principle

Tools are used to execute analysis, not define it.

The focus of this course is on:

* understanding problems
* choosing the right approach
* interpreting results for decision-making

---

## Notes

* You only need to use one tool at a time (Python or R)
* Labs are provided in both formats for flexibility
* Choose the tool you are more comfortable with
