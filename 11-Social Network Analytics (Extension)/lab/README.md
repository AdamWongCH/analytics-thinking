# Lab: Network Analysis

---

## Objective

Apply basic network analysis techniques.

---

## Tasks

You will:

* construct a network from interaction data
* visualise the network
* compute basic metrics (e.g. degree)
* interpret the results

---

## Deliverable

* Identify key nodes
* Describe network structure
* Suggest possible actions

---

## Reminder

Focus on interpretation, not just computation.
