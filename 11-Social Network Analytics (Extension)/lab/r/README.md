# R Lab

## Setup

Before running the lab, install the required package:

```r
install.packages("igraph")
```

Then load it:

```r
library(igraph)
```

Installation is only required once.

---

Open `social_network_analytics.R` in RStudio.

Run the script line by line using:

Ctrl + Enter

Work from top to bottom and pause at each section to interpret the output.
