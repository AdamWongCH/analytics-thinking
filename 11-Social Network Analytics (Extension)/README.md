# 11. Social Network Analytics (Extension)

---

## Learning Objectives

By the end of this module, you should be able to:

* understand how relationships can be represented as networks
* identify key nodes and connections in a network
* interpret simple network metrics
* apply network thinking to real-world problems

---

## Why This Module Matters

So far, we have analysed data in the form of:

* rows and columns
* variables and features

But many real-world problems involve relationships.

For example:

* who interacts with whom
* how information flows
* how influence spreads

Social Network Analytics (SNA) helps us understand:

> **how entities are connected**

---

## The Core Idea

Instead of analysing individual data points, we analyse:

* **nodes** → entities (e.g. students)
* **edges** → relationships (e.g. interactions)

This allows us to move from:

> analysing individuals → analysing relationships

---

## Scenario

You are given student interaction data from discussion forums.

Your task is to:

* identify highly connected students
* detect isolated students
* understand how engagement spreads
